# Week 1 — Project Setup & Requirements

- Defined project scope: decentralized healthcare platform for patient-controlled medical records.
- Created project repository on GitHub with initial README and MIT license.
- Set up monorepo structure with `/backend`, `/frontend`, `/contracts`, and `/docs` directories.
- Researched existing EHR systems (Epic, Cerner, OpenMRS) to identify pain points and opportunities.
- Drafted functional and non-functional requirements document covering user roles, data flow, and security constraints.
- Selected technology stack: Node.js/Express for backend, React/Vite for frontend, Solidity for smart contracts, Polygon Mumbai for blockchain, IPFS for storage.
