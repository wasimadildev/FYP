# Week 3 — Backend Authentication & User Management

- Implemented user registration with bcrypt password hashing (12 salt rounds).
- Built JWT authentication with access tokens (24h expiry) and refresh token rotation.
- Created role-based authorization middleware for patient, doctor, and admin routes.
- Added email verification flow using Nodemailer with verification tokens.
- Built admin user verification endpoints for approving new doctor registrations.
- Wrote unit tests for auth routes achieving 90% coverage on login/register flows.
