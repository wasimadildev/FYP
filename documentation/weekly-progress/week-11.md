# Week 11 — Deployment & Infrastructure

- Containerized backend and frontend with Docker and Docker Compose.
- Deployed backend API on Railway.app with auto-deploy from GitHub branch.
- Deployed frontend on Vercel with environment variables for API URL and contract addresses.
- Set up MongoDB Atlas production cluster with encryption at rest enabled.
- Configured IPFS pinning service via Pinata for reliable file persistence.
- Set up GitHub Actions CI pipeline for automated linting, testing, and deployment.
- Added Sentry error tracking for production monitoring.
